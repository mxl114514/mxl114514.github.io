# 梦珞计算器 Java版

基于 Java Swing 的桌面计算器，功能对齐 Python 版。

## 功能

- 基本四则运算（加、减、乘、除）
- 百分比计算
- 正负号切换
- 倒数、平方、开方
- 支持键盘输入
- 深色主题 UI

## 运行

需要 Java 17+：

```bash
java -jar Calculator.jar
```

或从源码编译：

```bash
javac Calculator.java
java Calculator
```

## 技术栈

- Java 17
- Swing (Javax)
